Discovery Projects Examples

Creating a new discovery project​

1. Head over to Discovery Projects Page from the Dashboard Sidebar.​

2. Enter Name Project goals and upload file for building the model.​

3. Obtain the Data Visualization done by the platform.​

4. Edit the data according to data cleaning standards.​

5. Select Parameters for the model and Train the model.​

6. Get the model results.​
