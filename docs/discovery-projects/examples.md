## Creating a new discovery project

### 1. Head over to Discovery Projects Page from the Dashboard Sidebar.
![Dashboard-to-Discovery](../../static/img/tutorials/discovery-projects/dashboard-to-discovery.png)
### 2. Enter Name Project goals and upload file for building the model.
![Discovery-Create-New](../../static/img/tutorials/discovery-projects/discovery-create-new.png)
### 3. Obtain the Data Visualization done by the platform.
![Discovery-Visualize-Data](../../static/img/tutorials/discovery-projects/discovery-visualize-data.png)
### 4. Edit the data according to data cleaning standards.
![Discovery-Transform-Data](../../static/img/tutorials/discovery-projects/discovery-transform-data.png)
### 5. Select Parameters for the model and Train the model.
![Discovery-Select-Params-and-Build-Model](../../static/img/tutorials/discovery-projects/discovery-select-params-and-build-model.png)
### 6. Get the model results.
![Discovery-Result](../../static/img/tutorials/discovery-projects/discovery-result.jpg)
